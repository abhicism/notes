# Notes Desktop App

A lightweight desktop note-taking application built with Python and Tkinter.

## Features

- Create notes with a title and rich multi-line body text.
- Search notes instantly by title or content.
- Auto-save all edits to local storage (`notes_data.json`).
- Delete notes with confirmation.
- Shows created and updated timestamps for the selected note.

## Run

```bash
python3 app.py
```

## Requirements

- Python 3.10+
- Tkinter (usually included with standard Python installs)

## Data Storage

Notes are stored in `notes_data.json` in the project directory.
